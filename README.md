# DocuMind-AI

Cloud-Native AI Document Processing Platform built using FastAPI, Google Gemini AI, Supabase, pgvector, Docker, and Railway.

## Features

- Resume Parsing
- Invoice Extraction
- Semantic Search
- AI-Powered Job Matching
- Recruiter Dashboard
- Platform Analytics
- Vector Search
- Cloud Deployment

## Tech Stack

- Google Gemini
- Supabase
- PostgreSQL
